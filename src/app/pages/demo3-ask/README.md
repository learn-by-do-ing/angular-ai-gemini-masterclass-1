# Demo 3: Service Architecture & Multimodal

## Description
Refactoring for scalability and introducing multimodal capabilities.

## Details
Moves the AI interaction logic into a dedicated `GeminiService`, promoting separation of concerns. It adds support for multimodal input (Text + Image) and parses the Markdown response for better presentation.

## Keypoint
Service-based architecture and Multimodal (Text + Image) support.
