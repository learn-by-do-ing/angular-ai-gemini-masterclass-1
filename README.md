# Angular Gemini Masterclass 1

A project with some demos to understand how to use Gemini in an Angular app.

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.0.4.

## Requirements

Before run the project you need to create the environment files in `src/environments` folder.

* `src/environments/environment.ts` for development
* `src/environments/environment.prod.ts` for production

Both files should have the same content:

```typescript
export const environment = {
  apiKey: 'YOUR_GEMINI_API_KEY'
};
```

## Run Project

To start a local development server, run:

```bash
npm start
```
